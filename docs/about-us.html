ABOUT US

GeniXCMS is a Free and Opensource Content Management System Which built with simplicity and easy to customize purpose. GeniXCMS initially was built on September, 2014. 

As the tremendously progress of web technology, there are a lot of Content Management with a complex and hard to customize. And to solve that problem, I started to built GeniXCMS from scratch. With a modern style of code but with a simplicity in mind so most people can understand it.  

By using GeniXCMS I hope there will be an opportunity to create a great website or web based applications. 
It's still immature but GeniXCMS was developed very well to fit with what people needs.
